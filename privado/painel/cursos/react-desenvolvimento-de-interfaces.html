Sobre o Curso

Aprenda a construir interfaces de usuário interativas e responsivas com React. Este curso abrangente foi projetado para desenvolvedores que desejam dominar a biblioteca React e criar aplicações web modernas com foco na experiência do usuário.

Ao longo do curso, você aprenderá a trabalhar com componentes, estados, props, hooks, gerenciamento de estado global, roteamento e integração com APIs. Você desenvolverá projetos práticos que simulam situações reais do mercado, preparando-o para construir aplicações profissionais em React.

Carlos Mendes

Desenvolvedor Frontend Sênior especializado em React, com experiência em grandes empresas de tecnologia.

O que você vai aprender

  • Fundamentos do React e JSX
  • Componentes funcionais e de classe
  • React Hooks (useState, useEffect, useContext, etc.)
  • Gerenciamento de estado com Redux
  • Roteamento com React Router
  • Estilização em React (CSS Modules, Styled Components)
  • Consumo de APIs e manipulação de dados
  • Testes em aplicações React
